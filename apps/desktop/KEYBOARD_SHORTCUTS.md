# RowFlow Keyboard Shortcuts Reference

## General

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open command palette |
| `Esc` | Close command palette |
| `↑` `↓` | Navigate commands |
| `Enter` | Execute selected command |

## Query Operations

| Shortcut | Action |
|----------|--------|
| `⌘Enter` / `Ctrl+Enter` | Run query |
| `⌥⌘F` / `Alt+Ctrl+F` | Format SQL |
| `⌘T` / `Ctrl+T` | New query tab |
| `⌘W` / `Ctrl+W` | Close query tab |

## Navigation

| Shortcut | Action |
|----------|--------|
| `⌘1` / `Ctrl+1` | Focus connections panel |
| `⌘2` / `Ctrl+2` | Focus schema browser |
| `⌘3` / `Ctrl+3` | Focus query editor |

## Connection

| Shortcut | Action |
|----------|--------|
| `⌘O` / `Ctrl+O` | Connect to database |
| `⌘D` / `Ctrl+D` | Disconnect |
| `⌘N` / `Ctrl+N` | New connection |

## Export

| Shortcut | Action |
|----------|--------|
| `⌘E` / `Ctrl+E` | Export to CSV |
| `⌘J` / `Ctrl+J` | Export to JSON |
| `⌘C` / `Ctrl+C` | Copy results |

## Help

| Shortcut | Action |
|----------|--------|
| `⌘?` / `Ctrl+?` | Show keyboard shortcuts |

## Pro Tips

### Command Palette Workflow

1. Press `⌘K` to open
2. Start typing to search
3. Use `↑` `↓` to navigate
4. Press `Enter` to execute
5. Press `Esc` to close

### Quick Access

- **Recent Commands**: Appear at the top of the palette
- **Fuzzy Search**: Type partial words to find commands
- **Category Browse**: Scroll to see all categories

### Common Patterns

**Query Workflow**:
1. `⌘K` → type "new query" → `Enter` (open new tab)
2. Write your SQL
3. `⌘Enter` (run query)
4. `⌘E` (export results)

**Connection Workflow**:
1. `⌘K` → type "connect" → `Enter`
2. Select profile
3. `⌘2` (focus schema browser)

**Navigation Workflow**:
- `⌘1` → Connections
- `⌘2` → Schema
- `⌘3` → Editor
- `⌘K` → Command palette (from anywhere)

## Platform-Specific Notes

### macOS
- Use `⌘` (Command key) for all shortcuts
- Use `⌥` (Option key) for Format SQL

### Windows/Linux
- Use `Ctrl` for all shortcuts
- Use `Alt` for Format SQL

## Accessibility

All commands are accessible via:
- Keyboard shortcuts
- Command palette search
- Mouse/touch interaction

The command palette provides keyboard-first navigation, making RowFlow fully accessible without a mouse.
